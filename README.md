# user-github-api

User GitHub Api created to Shaw and Partners admission test.

### Stack

| Responsability  | What |
|---|---|
| Process Managament  | PM2  |
| Code Standard  | ESLint  |    
| HTTP Headers  | Helmet |
| Test Runner | Mocha |
| CORS  | CORS |   
| Body Parsing  | Body Parser |   
| Versioning  | Git |
| Schema Validation | Joi |
| Gzip compression | Compression |
